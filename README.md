# nodered-read-csv
Node permettant de lire un tableau CSV

Pré-requis : 
  1. Node-Red
  2. un fichier CSV (voir exemple)

Utilisation :
 1. Placer le fichier CSV à un emplacement accessible par node-red
 2. Importer le fichier json en exemple
 3. Vérifier dans le degug
